# 基于React的移动端电子商城项目

学习前端时所写的练手项目。

技术栈：**create-react-app** + **React Hooks** + **Typescript**


预览地址：[电子商城](http://119.23.56.56:10000/)
## 特性
- 基于**vw** + **rem** + **flex** + **媒体查询**的移动端适配方案
- 全面采用函数式组件，并对常用功能封装**自定义Hooks**
- **React Hooks**实现**轮播图**

## 功能
- 商品列表
- 用户信息
- 购物车
- 商品购买
- 发布评论
  
## 安装及快速开始
```
npm instal
npm run start
```
